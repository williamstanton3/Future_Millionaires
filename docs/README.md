# Schedule Planner — Getting Started

## How to Run

### Step 1: Start the Backend Server

Open and run the following file in IntelliJ:

```
\Future_Millionaires\backend\src\main\java\edu\gcc\future_millionaires\Main.java
```

This starts the Javalin backend server on port 7070. Wait until you see confirmation in the IntelliJ console that the server is running before moving on.

### Step 2: Navigate to the Frontend

Open a terminal and change directory to the frontend folder:

```bash
cd "\Future_Millionaires\frontend"
```

### Step 3: Start the Frontend

Run the following command:

```bash
npm run dev
```

> If this fails, you may need to install dependencies first:
> ```bash
> npm install
> ```
> Then run `npm run dev` again.

### Step 4: Open the App

Click the localhost link printed in the terminal, or navigate to:

```
http://localhost:5173
```

### Step 5: You're good to go!

The app should now be fully functional with all the mvp requirements. The backend and frontend are both running and connected.
